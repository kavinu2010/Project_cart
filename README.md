<h1>Project Cart</h1>

Project Cart is a dynamic and user-friendly online shopping cart application designed to enhance your e-commerce experience.
Built with modern web technologies, it allows users to browse products, add items to their cart, and manage their selections seamlessly across various devices.

<h2>Features</h2>

<li>Browse Products: View a variety of products fetched from a reliable API.</li>
 
<li>Add to Cart: Easily add desired products to your shopping cart with a single click.</li>
<li>Remove from Cart: Remove items from your cart effortlessly.</li>
<li>Responsive Design: Fully responsive layout ensuring optimal viewing on desktops, tablets, and mobile devices.</li>
<li>User-Friendly Interface: Clean and intuitive UI for a seamless shopping experience.</li>
<li>Real-Time Cart Count: Dynamic cart item count updates in real-time as you add or remove products.</li>
<li>Toggle Cart View: Easily switch between product listings and your cart.</li>



<h2>Technologies Used
</h2>



HTML5: Structuring the content and layout of the web pages.
CSS3: Styling the application to ensure a visually appealing and responsive design.
JavaScript (ES6): Adding interactivity and dynamic functionality to the application.
Fetch API: Retrieving product data from an external API.
DummyJSON API: Source of dummy product data for demonstration purposes.
